Section 01

System Overview & Design Philosophy

PRISM wraps a Uniswap V4 pool with a vault contract that refracts LP capital into a configurable spectrum of tick-range positions. Where Uniswap V3 made liquidity provision capital-efficient but operationally brutal, and where active managers like Gamma V2 smoothed out the day-to-day but remained bottlenecked by centralized advisors, PRISM uses V4's hook primitive and flash-accounting to push the entire management loop on-chain: deposit once, rebalance atomically, charge volatility-adaptive fees, and recapture MEV that would otherwise bleed to external arbitrageurs.

The system is designed for a 7-day solo build sprint with Claude as a development partner. Each day delivers a fully functional, testable layer — no day depends on unfinished work from the next. The sprint culminates in a public Base Sepolia testnet launch on Day 7, with a capped-deposit soft-launch path available for Base mainnet once the testnet run has stabilized for 1–2 weeks.

Core Design Principles

Refraction as a Metaphor

One LP deposit is refracted into N tick-range positions via a pluggable IStrategy. Bell, exponential, and order-book shapes are three spectrums of the same underlying capital.

Custody-Free Automation

No advisor, no admin-held keys on user funds. Rebalances are keeper-triggered permissionlessly; the strategy decides whether they happen. Users retain full withdrawal rights at all times.

Hook-Native from Day One

PRISM is not retrofitted onto V3-style liquidity. Dynamic fees via beforeSwap, MEV capture via afterSwap, and flash-accounted atomic rebalances are intrinsic to the design.

Immutable Core

No upgrade proxies in v1. If a critical bug is found, contracts are redeployed and users migrate. This removes governance-as-attack-surface in exchange for upfront rigor.

Solo-Buildable

Scope is cut to what a solo dev can ship in 7 days without cutting corners on security. Advanced features (MEV execution, indexer, multi-chain, multi-strategy) are post-sprint, not hidden debt.

Composable ERC-20

Vault shares are standard ERC-20 — transferable, tradeable, collateral-izable. No transfer hooks, no fees on shares, no opinions about what downstream protocols do with them.

Section 02

High-Level System Architecture

The system is organized into four layers: the user layer (LPs, swappers, keepers), the presentation layer (Next.js dApp + wagmi), the protocol layer (Vault, Strategy, Hook contracts), and the Uniswap V4 layer (PoolManager singleton). Every vault operation is an unlock-callback into PoolManager; every swap through a PRISM-hooked pool routes through the hook's beforeSwap and afterSwap callbacks. There is no off-chain state in the critical path.

Hot-Path Sequence: Swap Through a PRISM-Hooked Pool

Every swap against a PRISM-hooked pool triggers the full hook lifecycle. Dynamic fee computation occurs pre-swap; MEV observation and optional backrun occurs post-swap. Both are bounded operations with gas budgets.

Hook Gas Budget

beforeSwap: ≤ 12k gas — EWMA update (3 SSTOREs), fee calc (arithmetic + clamp), return. afterSwap (MVP): ≤ 15k gas — oracle read (1 SLOAD + 1 external call), deviation check, event emit. afterSwap (v1.1 with backrun): ≤ 80k gas — oracle + optional swap + accounting. Total hook overhead per swap: ~25k gas MVP, ~90k with active MEV capture — acceptable margin vs. expected backrun yield.

Atomic Rebalance via Flash Accounting

The entire rebalance — remove all positions, collect fees, internal swap to target ratio, deploy new positions — executes inside a single PoolManager.unlock() callback. EIP-1153 transient storage tracks the currency deltas; tokens only physically transfer when the net delta settles at the end. A N-position rebalance touches each token balance exactly once, not 2N times.

Core Interfaces & Errors

Every contract in PRISM speaks through these three interfaces. Defining them on Day 1 — even before implementations exist — forces us to think in terms of contracts, not functions. The Errors library centralizes custom error types so every revert costs ~200 gas (selector only) instead of the 2000+ of string reverts.

packages/contracts/src/interfaces/IVault.sol 
// SPDX-License-Identifier: BUSL-1.1
pragma solidity ^0.8.25;

import {IERC20} from "openzeppelin-contracts/token/ERC20/IERC20.sol";
import {PoolKey} from "v4-core/types/PoolKey.sol";

/// @notice Vault interface — multi-position LP aggregator on top of a V4 pool
interface IVault is IERC20 {
    /// @notice Single tick-range position tracked by the vault
    struct Position {
        int24 tickLower;
        int24 tickUpper;
        uint128 liquidity;
    }

    event Deposit(address indexed user, uint256 amount0, uint256 amount1, uint256 shares);
    event Withdraw(address indexed user, uint256 amount0, uint256 amount1, uint256 shares);
    event Rebalanced(int24 tick, uint256 nPositions, uint256 gasUsed);
    event FeesCollected(uint256 fees0, uint256 fees1);

    /// @notice Deposit both tokens, receive ERC-20 shares proportional to deposit value.
    /// @dev    Reverts if amount < mins or if deposit would exceed TVL cap.
    function deposit(
        uint256 amount0Desired,
        uint256 amount1Desired,
        uint256 amount0Min,
        uint256 amount1Min,
        address to
    ) external returns (uint256 shares, uint256 amount0, uint256 amount1);

    /// @notice Burn shares to withdraw proportional share of all positions + idle balances.
    /// @dev    Always callable, even when deposits are paused.
    function withdraw(
        uint256 shares,
        uint256 amount0Min,
        uint256 amount1Min,
        address to
    ) external returns (uint256 amount0, uint256 amount1);

    /// @notice Permissionless rebalance trigger. Reverts if strategy.shouldRebalance() is false.
    function rebalance() external;

    /// @notice Current positions held by the vault
    function getPositions() external view returns (Position[] memory);

    /// @notice Total underlying assets (idle balances + in-range liquidity value)
    function getTotalAmounts() external view returns (uint256 total0, uint256 total1);

    /// @notice Pool this vault provides liquidity to
    function poolKey() external view returns (PoolKey memory);
}

        int24 tickLower;

ERC-20 Shares & Value Accounting

Vault shares are minted pro-rata to deposit value relative to total vault value. Total value includes: idle token balances, current value of all in-range liquidity positions at the active tick, and accrued-but-uncollected swap fees. Using Solmate's ERC-20 base gets us efficient storage and permit for free.

packages/contracts/src/core/Vault.sol — deposit (abridged) 
// SPDX-License-Identifier: BUSL-1.1
pragma solidity ^0.8.25;

import {ERC20} from "solmate/tokens/ERC20.sol";
import {SafeTransferLib} from "solmate/utils/SafeTransferLib.sol";
import {IPoolManager} from "v4-core/interfaces/IPoolManager.sol";
import {PoolKey} from "v4-core/types/PoolKey.sol";
import {FullMath} from "v4-core/libraries/FullMath.sol";
import {Errors} from "../utils/Errors.sol";
import {ReentrancyGuardTransient} from "../utils/ReentrancyGuardTransient.sol";

contract Vault is ERC20, ReentrancyGuardTransient {
    using SafeTransferLib for ERC20;

    uint256 public constant MAX_POSITIONS = 30;
    uint256 public constant MIN_SHARES    = 1_000; // burned to DEAD on first deposit
    address public constant DEAD          = 0x000000000000000000000000000000000000dEaD;

    IPoolManager public immutable poolManager;
    IStrategy    public immutable strategy;
    PoolKey      public poolKey;
    uint256      public tvlCap;
    bool         public depositsPaused;

    Position[] public positions;
    int24      public lastRebalanceTick;
    uint256    public lastRebalanceTimestamp;

    struct Position {
        int24   tickLower;
        int24   tickUpper;
        uint128 liquidity;
    }

    // ... constructor, modifiers omitted ...

    function deposit(
        uint256 amount0Desired,
        uint256 amount1Desired,
        uint256 amount0Min,
        uint256 amount1Min,
        address to
    ) external nonReentrant returns (uint256 shares, uint256 amount0, uint256 amount1) {
        if (depositsPaused) revert Errors.DepositsPaused();

        _collectFees();
        (uint256 total0, uint256 total1) = getTotalAmounts();

        uint256 _totalSupply = totalSupply;
        if (_totalSupply == 0) {
            // First depositor: seed shares equal to geometric mean, burn MIN_SHARES
            shares = FullMath.mulDiv(amount0Desired, amount1Desired, 1e18).sqrt();
            if (shares <= MIN_SHARES) revert Errors.ZeroShares();
            _mint(DEAD, MIN_SHARES);
            shares -= MIN_SHARES;
            amount0 = amount0Desired;
            amount1 = amount1Desired;
        } else {
            // Subsequent: pro-rata to total vault value
            uint256 shares0 = total0 == 0 ? type(uint256).max
                : FullMath.mulDiv(amount0Desired, _totalSupply, total0);
            uint256 shares1 = total1 == 0 ? type(uint256).max
                : FullMath.mulDiv(amount1Desired, _totalSupply, total1);
            shares = shares0 < shares1 ? shares0 : shares1;
            amount0 = total0 == 0 ? amount0Desired
                : FullMath.mulDiv(shares, total0, _totalSupply);
            amount1 = total1 == 0 ? amount1Desired
                : FullMath.mulDiv(shares, total1, _totalSupply);
        }

        if (amount0 < amount0Min || amount1 < amount1Min) {
            revert Errors.SlippageExceeded(amount0 < amount0Min ? amount0 : amount1,
                                            amount0 < amount0Min ? amount0Min : amount1Min);
        }
        if (total0 + amount0 > tvlCap) revert Errors.TVLCapExceeded(total0 + amount0, tvlCap);

        ERC20(Currency.unwrap(poolKey.currency0)).safeTransferFrom(msg.sender, address(this), amount0);
        ERC20(Currency.unwrap(poolKey.currency1)).safeTransferFrom(msg.sender, address(this), amount1);

        _deployLiquidity(amount0, amount1); // unlock + modifyLiquidity across positions

        _mint(to, shares);
        emit Deposit(msg.sender, amount0, amount1, shares);
    }
}

IStrategy — Final Form

Two functions, zero mutable state. Strategies are pure-ish: they read the tick and the available amounts, and return a deterministic list of positions with normalized weights.

packages/contracts/src/strategies/BellStrategy.sol 
// SPDX-License-Identifier: MIT
pragma solidity ^0.8.25;

import {IStrategy}   from "../interfaces/IStrategy.sol";
import {PositionLib} from "../libraries/PositionLib.sol";
import {Errors}      from "../utils/Errors.sol";

/// @notice Bell-shaped liquidity distribution centered on current tick.
///         Weights decay symmetrically toward the range edges (Gaussian approximation).
contract BellStrategy is IStrategy {
    uint256 public immutable spreadTicks;        // total width in ticks
    uint256 public immutable numPositions;       // N = positions to generate (odd recommended)
    uint256 public immutable deviationThreshold; // ticks of drift before rebalance
    uint256 public immutable timeFallback;       // seconds since last rebalance before forced

    uint256 private constant WEIGHT_DENOMINATOR = 10_000;

    constructor(uint256 _spread, uint256 _n, uint256 _dev, uint256 _time) {
        require(_n > 0 && _n <= 30, "bad N");
        require(_spread > 0, "bad spread");
        spreadTicks        = _spread;
        numPositions       = _n;
        deviationThreshold = _dev;
        timeFallback       = _time;
    }

    function computePositions(
        int24 currentTick,
        int24 tickSpacing,
        uint256 /* amount0 */,
        uint256 /* amount1 */
    ) external view returns (TargetPosition[] memory positions) {
        int24 center    = PositionLib.roundTickDown(currentTick, tickSpacing);
        int24 halfRange = int24(int256(spreadTicks / 2));
        int24 step      = int24(int256(spreadTicks / numPositions));

        positions = new TargetPosition[](numPositions);
        uint256 totalW;

        for (uint256 i; i < numPositions; ) {
            int24 lower = center - halfRange + int24(int256(i)) * step;
            int24 upper = lower + step;

            // Align to tick spacing
            lower = PositionLib.roundTickDown(lower, tickSpacing);
            upper = PositionLib.roundTickDown(upper, tickSpacing);
            if (upper == lower) upper += tickSpacing;

            // Gaussian-approx weight: triangular peak at center
            int256 mid      = int256(lower + upper) / 2;
            int256 dist     = mid > int256(center) ? mid - int256(center) : int256(center) - mid;
            uint256 absDist = uint256(dist);
            uint256 halfU   = uint256(uint24(halfRange));
            uint256 weight  = absDist >= halfU ? 0
                : WEIGHT_DENOMINATOR - (absDist * WEIGHT_DENOMINATOR / halfU);

            positions[i] = TargetPosition({
                tickLower: lower,
                tickUpper: upper,
                weight:    weight
            });
            totalW += weight;

            unchecked { ++i; }
        }

        // Normalize so Sum(weight) = 10_000 exactly
        if (totalW == 0) revert Errors.WeightsDoNotSum(0);
        uint256 acc;
        for (uint256 i; i < numPositions; ) {
            if (i == numPositions - 1) {
                // Last position: assign remainder to guarantee exact sum
                positions[i].weight = WEIGHT_DENOMINATOR - acc;
            } else {
                positions[i].weight = positions[i].weight * WEIGHT_DENOMINATOR / totalW;
                acc += positions[i].weight;
            }
            unchecked { ++i; }
        }
    }

    function shouldRebalance(
        int24 currentTick,
        int24 lastRebalanceTick,
        uint256 lastRebalanceTimestamp
    ) external view returns (bool) {
        int256 drift     = int256(currentTick) - int256(lastRebalanceTick);
        uint256 absDrift = drift >= 0 ? uint256(drift) : uint256(-drift);
        if (absDrift > deviationThreshold) return true;
        if (block.timestamp - lastRebalanceTimestamp > timeFallback) return true;
        return false;
    }
}

function unlockCallback(bytes calldata data) external override returns (bytes memory) {
    if (msg.sender != address(poolManager)) revert Errors.OnlyPoolManager();
    Action action = abi.decode(data, (Action));

    if (action == Action.REBALANCE) {
        (,int24 currentTick,,) = poolManager.getSlot0(_poolId());

        // 1. Remove all existing positions

        for (uint256 i; i < n; ) {
            Position memory p = positions[i];
            if (p.liquidity > 0) {
                poolManager.modifyLiquidity(
                    poolKey,
                    IPoolManager.ModifyLiquidityParams({
                        tickLower:      p.tickLower,
                        tickUpper:      p.tickUpper,
                        liquidityDelta: -int256(uint256(p.liquidity)),
                        salt:           bytes32(0)
                    }),
                    ""
                );
            }
            unchecked { ++i; }
        }

        // 2. Compute new target positions
        (uint256 total0, uint256 total1) = _getTotalAssetsFromDeltas();
        IStrategy.TargetPosition[] memory targets =
            strategy.computePositions(currentTick, poolKey.tickSpacing, total0, total1);

        // 3. Internal swap to rebalance ratio (slippage-bounded)
        (uint256 need0, uint256 need1) = _computeRequiredAmounts(targets, currentTick);
        if (total0 > need0 + DUST) _swapExcess(true,  total0 - need0);
        else if (total1 > need1 + DUST) _swapExcess(false, total1 - need1);

        // 4. Deploy new positions
        delete positions;
        for (uint256 i; i < targets.length; ) {
            uint128 liq = _liquidityForWeight(targets[i], total0, total1, currentTick);
            if (liq > 0) {
                poolManager.modifyLiquidity(
                    poolKey,
                    IPoolManager.ModifyLiquidityParams({
                        tickLower:      targets[i].tickLower,
                        tickUpper:      targets[i].tickUpper,
                        liquidityDelta: int256(uint256(liq)),
                        salt:           bytes32(0)
                    }),
                    ""
                );
                positions.push(Position({
                    tickLower: targets[i].tickLower,
                    tickUpper: targets[i].tickUpper,
                    liquidity: liq
                }));
            }
            unchecked { ++i; }
        }

        // 5. Settle all deltas — PoolManager handles net transfer
        _settleDeltas();

        lastRebalanceTick      = currentTick;
        lastRebalanceTimestamp = block.timestamp;
        emit Rebalanced(currentTick, positions.length, 0);
    }

    return "";
}

/// @notice Off-chain helper: deterministic CREATE2 salt search for a hook address
///         with required permission bit flags set. Used in deploy scripts only.
library HookMiner {
    uint160 internal constant FLAG_MASK = 0x3FFF; // lsb 14 bits

    function find(
        address deployer,
        uint160 requiredFlags,
        bytes memory creationCode,
        bytes memory constructorArgs
    ) internal pure returns (address hookAddress, bytes32 salt) {
        bytes memory codeWithArgs = abi.encodePacked(creationCode, constructorArgs);
        bytes32 codeHash          = keccak256(codeWithArgs);

        for (uint256 i; i < 200_000; ) {
            salt        = bytes32(i);
            hookAddress = computeAddress(deployer, salt, codeHash);
            if ((uint160(hookAddress) & FLAG_MASK) == requiredFlags) return (hookAddress, salt);
            unchecked { ++i; }
        }
        revert("HookMiner: no salt found in 200k iterations");
    }

    function computeAddress(address deployer, bytes32 salt, bytes32 codeHash)
        internal pure returns (address)
    {
        return address(uint160(uint256(keccak256(
            abi.encodePacked(bytes1(0xff), deployer, salt, codeHash)
        ))));
    }
}

beforeSwap — Volatility-Adaptive Fee

packages/contracts/src/libraries/FeeLib.sol 
// SPDX-License-Identifier: MIT
pragma solidity ^0.8.25;

library FeeLib {
    uint24  internal constant MIN_FEE       = 100;       // 0.01%
    uint24  internal constant MAX_FEE       = 100_000;   // 10%
    uint24  internal constant BASE_FEE      = 3_000;     // 0.30% (V3 standard)
    uint256 internal constant PRECISION     = 1e18;
    uint256 internal constant SHORT_ALPHA   = 5e16;      // 0.05 (~20-swap half-life)
    uint256 internal constant LONG_ALPHA    = 5e15;      // 0.005 (~200-swap half-life)

    struct VolatilityState {
        int24   lastTick;
        uint256 lastTimestamp;
        uint256 ewmaShort; // short-window variance (Q64.64-ish, 18 dec)
        uint256 ewmaLong;  // long-window variance (baseline)
    }

    /// @notice Update EWMA state on each swap
    function update(VolatilityState storage s, int24 currentTick) internal {
        int256 dt     = int256(currentTick) - int256(s.lastTick);
        uint256 sqr   = uint256(dt * dt) * PRECISION;
        s.ewmaShort   = (SHORT_ALPHA * sqr + (PRECISION - SHORT_ALPHA) * s.ewmaShort) / PRECISION;
        s.ewmaLong    = (LONG_ALPHA  * sqr + (PRECISION - LONG_ALPHA)  * s.ewmaLong)  / PRECISION;
        s.lastTick      = currentTick;
        s.lastTimestamp = block.timestamp;
    }

    /// @notice Dynamic fee = BASE_FEE * (ewmaShort / ewmaLong), clamped to [MIN, MAX]
    function calculate(VolatilityState storage s) internal view returns (uint24) {
        if (s.ewmaLong == 0) return BASE_FEE;
        uint256 ratio = s.ewmaShort * PRECISION / s.ewmaLong;
        uint256 raw   = uint256(BASE_FEE) * ratio / PRECISION;
        if (raw < MIN_FEE) return MIN_FEE;
        if (raw > MAX_FEE) return MAX_FEE;
        return uint24(raw);
    }
}

      /* to */ '0x...userAddress',
    ],
  });

  const { writeContract, data: hash, isPending } = useWriteContract();
  const { isLoading: isMining, isSuccess } =
    useWaitForTransactionReceipt({ hash });

  const deposit = () => sim?.request && writeContract(sim.request);

  return { deposit, simError, isPending, isMining, isSuccess, hash };
}

'use client';
import { useMemo } from 'react';
import * as d3 from 'd3-scale';
import type { Position } from '@/lib/types';

interface Props {
  positions: Position[];
  currentTick: number;
  width?: number;
  height?: number;
}

export function PrismVisual({ positions, currentTick, width = 640, height = 180 }: Props) {
  const { tickDomain, bars } = useMemo(() => {
    if (!positions.length) return { tickDomain: [0, 0], bars: [] };

    const minTick = Math.min(...positions.map(p => p.tickLower));
    const maxTick = Math.max(...positions.map(p => p.tickUpper));
    const pad     = Math.floor((maxTick - minTick) * 0.08);
    const domain  = [minTick - pad, maxTick + pad] as const;
    const x       = d3.scaleLinear().domain(domain).range([0, width]);
    const maxLiq  = Math.max(...positions.map(p => Number(p.liquidity)));
    const y       = d3.scaleLinear().domain([0, maxLiq]).range([height - 24, 4]);

    const bars = positions.map((p, i) => ({
      x:        x(p.tickLower),
      w:        x(p.tickUpper) - x(p.tickLower),
      y:        y(Number(p.liquidity)),
      h:        height - 24 - y(Number(p.liquidity)),
      key:      i,
      gradient: `spec-${(i % 6) + 1}`,
    }));

    return { tickDomain: domain, bars };
  }, [positions, width, height]);

  const cursorX = currentTick != null
    ? d3.scaleLinear().domain(tickDomain).range([0, width])(currentTick)
    : null;

  return (
    <svg viewBox={`0 0 ${width} ${height}`} className="w-full">
      <defs>
        {[1,2,3,4,5,6].map(i => (
          <linearGradient key={i} id={`spec-${i}`} x1="0" x2="0" y1="0" y2="1">
            <stop offset="0%"   stopColor={`var(--spec-${i})`} stopOpacity="0.9"/>
            <stop offset="100%" stopColor={`var(--spec-${i})`} stopOpacity="0.15"/>
          </linearGradient>
        ))}
      </defs>
      {bars.map(b => (
        <rect key={b.key} x={b.x} y={b.y} width={b.w} height={b.h}
              fill={`url(#${b.gradient})`} rx="1" />
      ))}
      {cursorX != null && (
        <line x1={cursorX} x2={cursorX} y1={0} y2={height - 20}
              stroke="var(--ink-bright)" strokeWidth="1" strokeDasharray="2 3" />
      )}
    </svg>
  );
}

TypeScript Keeper Bot

The keeper is a single TypeScript file using viem. It polls every 30s, asks strategy.shouldRebalance() via eth_call, simulates vault.rebalance(), and submits if the simulation succeeds. Deploys to Fly.io as a long-running process.

apps/keeper/src/index.ts 
import { createPublicClient, createWalletClient, http, getContract } from 'viem';
import { privateKeyToAccount } from 'viem/accounts';
import { baseSepolia } from 'viem/chains';
import { vaultAbi, strategyAbi, VAULTS } from './config';

const RPC = process.env.BASE_SEPOLIA_RPC!;
const PK  = process.env.KEEPER_PK! as `0x${string}`;
const POLL_INTERVAL = 30_000;

const account = privateKeyToAccount(PK);
const publicClient = createPublicClient({ chain: baseSepolia, transport: http(RPC) });
const walletClient = createWalletClient({ account, chain: baseSepolia, transport: http(RPC) });

async function checkAndRebalance(vaultAddr: `0x${string}`) {
  const vault    = getContract({ address: vaultAddr, abi: vaultAbi, client: publicClient });
  const strategy = await vault.read.strategy();
  const strat    = getContract({ address: strategy, abi: strategyAbi, client: publicClient });

  const [currentTick] = await vault.read.currentTick();
  const lastTick      = await vault.read.lastRebalanceTick();
  const lastTs        = await vault.read.lastRebalanceTimestamp();

  const should = await strat.read.shouldRebalance([currentTick, lastTick, lastTs]);
  if (!should) { console.log(`[${vaultAddr}] no rebalance needed`); return; }

  // Simulate first
  try {
    const { request } = await publicClient.simulateContract({
      account, address: vaultAddr, abi: vaultAbi, functionName: 'rebalance'
    });
    const hash = await walletClient.writeContract(request);
    console.log(`[${vaultAddr}] rebalance submitted: ${hash}`);
    const rc = await publicClient.waitForTransactionReceipt({ hash });
    console.log(`[${vaultAddr}] rebalance confirmed in block ${rc.blockNumber}`);
  } catch (e) {
    console.error(`[${vaultAddr}] simulation failed, skipping:`, e);
  }
}

async function loop() {
  while (true) {
    for (const vault of VAULTS) {
      try { await checkAndRebalance(vault); }
      catch (e) { console.error(`Vault ${vault}:`, e); }
    }
    await new Promise(r => setTimeout(r, POLL_INTERVAL));
  }
}

loop().catch(e => { console.error('Fatal:', e); process.exit(1); });

import {Test} from "forge-std/Test.sol";

contract VaultInvariants is Test {
    VaultHandler handler;

    function setUp() public {
        handler = new VaultHandler();
        targetContract(address(handler));
    }

    /// Invariant 1: totalShares > 0 implies positive underlying assets
    function invariant_nonzeroSharesImplyNonzeroAssets() public view {
        uint256 supply = handler.vault().totalSupply();
        if (supply == 0) return;
        (uint256 t0, uint256 t1) = handler.vault().getTotalAmounts();
        assertTrue(t0 > 0 || t1 > 0, "shares outstanding but vault empty");
    }

    /// Invariant 2: positions.length never exceeds MAX_POSITIONS
    function invariant_positionsWithinCap() public view {
        assertLe(handler.vault().getPositions().length, 30);
    }

    /// Invariant 3: currency deltas settle to zero after every op
    function invariant_deltaSettlesToZero() public view {
        assertEq(handler.poolManager().currencyDelta(address(handler.vault()),
                 handler.vault().poolKey().currency0), 0);
        assertEq(handler.poolManager().currencyDelta(address(handler.vault()),
                 handler.vault().poolKey().currency1), 0);
    }

    /// Invariant 4: mevProfitsAccrued only grows until claim
    function invariant_mevMonotonic() public view {
        assertGe(handler.hook().mevProfitsAccrued(address(handler.vault())),
                 handler.lastObservedMEV());
    }

    /// Invariant 5: withdraw never reverts when user has shares
    function invariant_withdrawAlwaysSucceeds() public {
        // handler's withdrawRandomHolder() sets `lastWithdrawReverted`
        assertFalse(handler.lastWithdrawReverted(), "withdraw reverted");
    }

    // Invariants 6 and 7: share-price monotonicity, weight sum = 10_000

}

2% annualized on vault TVL, accrued per-block, deducted from share price. Activated only after mainnet launch + audit. Transparent and cap-able by governance.

When VaultFactory is opened to external creators, a fraction of that vault's performance fee routes to the creator. Aligns curation incentives with ecosystem growth.
